Проект 13: Разработка A/B-тестирования и анализ результатов.
Цель: Оценить, улучшает ли новый алгоритм вовлечённость пользователей. Основная метрика успеха.

Задачи:

- Определить базовую долю «глубоких» сессий (≥4 страниц) по историческим данным.
- Убедиться в отсутствии пересечений пользователей между группами.
- Сравнить основную метрику (долю сессий с page_counter ≥ 4) между контрольной и тестовой группами с помощью z-теста для пропорций.
- Проанализировать вторичные метрики (число сессий на пользователя, регистрации) и проверить отсутствие негативных эффектов.
  
Методы и инструменты: Python (Pandas, Matplotlib, Seaborn), статистические тесты (t-тест)

Результат:
Новый алгоритм рекомендаций оказал положительное влияние на ключевую метрику — долю успешных сессий.
