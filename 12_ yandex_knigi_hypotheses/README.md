Проект 12: Сравнение пользовательской активности в приложении «Яндекс Книги» (Москва vs Санкт-Петербург)
Цель: Выявить статистически значимые различия в среднем времени активности пользователей мобильного приложения для чтения и прослушивания аудиокниг между двумя крупнейшими городами России — Москвой и Санкт-Петербургом.

Задачи:

- Загрузить и объединить данные пользователей из Москвы и Санкт-Петербурга из файла yandex_knigi_data.csv.
- Сравнить размеры выборок и рассчитать основные описательные статистики (среднее, медиана, стандартное отклонение) для каждой группы.
- Провести односторонний t-тест для двух независимых выборок, чтобы проверить гипотезу о большем времени активности пользователей из Санкт-Петербурга.
- Интерпретировать полученные результаты и предложить возможные причины различий.
  
Методы и инструменты: Python (Pandas, Matplotlib, Seaborn), ETL, анализ временных рядов, статистическая сводка.

Результат:
Изменение, которое было внедрено в группу B, оказало положительное влияние на показатель конверсии покупателей. Ожидаемый эффект от изменения достигнут и превышает первоначальные ожидания.
