## **Проект 1: Исследование развития игровой индустрии (2000–2013)**
### Цель
Выявить ключевые тренды в игровой индустрии (топ-платформы, жанры, региональные предпочтения) для определения стратегии разработки RPG-игры.

### Задачи
- Провести предобработку данных (очистка, категоризация оценок, фильтрация по периоду)
- Выделить топ-7 платформ по количеству выпущенных игр
- Проанализировать объёмы продаж по жанрам и регионам

### Методы и инструменты
Python (Pandas, NumPy), предобработка данных, категоризация, описательная статистика

### Результат
Получен очищенный датасет (12 780 записей), выявлены топ-7 платформ и динамика продаж. Результаты легли в основу рекомендаций по фокусу на RPG-разработке для определённых платформ и регионов.

## **Проект 2: Анализ ДТП в Кировской и Московской областях**
### Цель
Выявить факторы, влияющие на аварийность, для разработки мер по повышению безопасности.

### Задачи
- Объединить данные из 6 источников (ДТП, участники, транспорт)
- Проанализировать динамику ДТП по дням недели, месяцам и стажу водителей
- Сравнить показатели аварийности между регионами

### Методы и инструменты
Python (Pandas, Matplotlib, Seaborn), ETL, анализ временных рядов, статистическая сводка

### Результат
Установлено, что пик ДТП приходится на пятницу/субботу и август. Относительная аварийность в Кировской области в 2 раза выше, чем в Московской. Сформулированы рекомендации для точечного улучшения инфраструктуры.
